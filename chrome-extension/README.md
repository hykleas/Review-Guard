# ReviewGuard Chrome Extension

Bu extension, Google Maps yorumlarını otomatik olarak doldurur.

## Kurulum

1. Chrome'da `chrome://extensions/` adresine gidin
2. Sağ üstte "Geliştirici modu"nu açın
3. "Paketlenmemiş uzantı yükle" butonuna tıklayın
4. Bu klasörü seçin (`chrome-extension`)

## Kullanım

1. ReviewGuard sisteminde yorum yazın
2. Sistem sizi Google Maps'e yönlendirir
3. Extension otomatik olarak yorumu doldurur
4. Sadece "Gönder" butonuna tıklayın

## Notlar

- Extension sadece Google Maps sayfalarında çalışır
- Yorum bilgileri localStorage'da saklanır
- 10 dakika sonra otomatik temizlenir
